# 📄 AI Resume Analyzer

An ATS-style AI Resume Analyzer built using Python and Streamlit that evaluates resumes, detects technical skills, compares resumes with job descriptions, and provides smart analytics.

---

## 🚀 Live Demo
https://ai-resume-analyzer-gwxthgqzdgrr9yssacddw4.streamlit.app/

---

## 🔥 Features

- Resume PDF parsing
- Skill extraction
- ATS-style resume scoring
- Job Description (JD) matching
- Missing skill detection
- Resume highlighting
- Interactive dashboard
- Downloadable analysis report
- Charts and analytics

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- PyPDF / PyPDF2
- Regular Expressions
- Data Visualization

---

## 📊 Dashboard Features

- Resume Score
- Skill Coverage
- JD Match Analysis
- Missing JD Keywords
- Skill Charts
- Resume Preview
